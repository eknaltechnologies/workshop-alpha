# 📌 Important Rules

## ✅ Create Your Own Folder Only

Every participant must create a folder using their full name.

### Example

```bash
Rahul_Kumar/
Ananya_Reddy/
John_Doe/
```

❌ Do NOT upload files outside your folder  
❌ Do NOT modify other participant folders  
❌ Do NOT delete other participant files

---

## ✅ Create Your Own Branch

Every participant must work on their own branch.

### Branch Naming Format

```bash
your-name
```

### Example

```bash
rahul-kumar
ananya-reddy
john-doe
```

❌ Do NOT push directly to `main` branch

All submissions must happen through:

- Your own branch
- Pull Request (PR)
- Review & Merge process

---

## ✅ Push Daily Updates

Participants should upload:

- Daily practice programs
- Assignments
- Tasks
- Notes
- Challenges
- Mini projects

Try to push your code daily after the workshop session.

---

## ✅ Maintain Proper Folder Structure

Create separate folders for each day.

### Example

```bash
Your_Name/
│
├── Day_01/
├── Day_02/
├── Day_03/
└── Mini_Project/
```

---

## ✅ Use Meaningful File Names

### Good Examples

```bash
variables.py
loops_practice.py
calculator_project.py
student_management.py
```

### Bad Examples

```bash
test.py
abc.py
new.py
practice1.py
```

---

## ✅ Write Clean & Readable Code

- Use proper indentation
- Write understandable code
- Add comments when necessary
- Avoid unnecessary code
- Keep your folders organized

---

## ❌ Avoid Uploading

Please DO NOT upload:

- `.exe` files
- Personal files
- Random screenshots
- Movies/music
- Large unnecessary files
- Virtual environments (`venv/`)
- Cache folders (`__pycache__/`)
- IDE settings folders

# ⚙️ Initial Git Setup

## 1️⃣ Install Git

Download Git from:

### Windows

https://git-scm.com/downloads

### Linux

```bash
sudo apt install git
```

### Mac

```bash
brew install git
```

---

# ⚙️ Configure Git

Run these commands once after installing Git:

```bash
git config --global user.name "Your Name"
git config --global user.email "yourmail@example.com"
```

### Example

```bash
git config --global user.name "Rahul Kumar"
git config --global user.email "rahul@gmail.com"
```

---

# ✅ Git Commands For Daily Submission

## Check Current Changes

```bash

```

---

## Add Files

```bash

```

---

## Commit Changes

```bash

```

### Examples



# 🔄 Create Pull Request (PR)

After pushing your code:

1. Open GitHub repository
2. GitHub will show:
   > Compare & Pull Request
3. Click it
4. Add proper PR title
5. Create Pull Request
6. Wait for review & merge
